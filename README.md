# AHAVA Landing (Vite + React + Tailwind v4)

Proyecto base listo para correr con Tailwind **v4** (sin PostCSS extra).

## Requisitos
- Node 18+
- pnpm 9/10 **o** npm (si tu pnpm tiene problemas)

## Instalar y correr

### Con pnpm
```bash
pnpm install
pnpm run dev
```

### Con npm (alternativa garantizada)
```bash
npm install
npm run dev
```

## Notas
- El video del hero se carga desde **/public/videos/ahava-hero.mp4**. Coloca tu archivo ahí (mismo nombre).
- El logo está en **/public/ahava-logo.png**.
- Tailwind v4 funciona con `@import "tailwindcss";` y el bloque `@theme` en `src/index.css`.
- No se requieren `tailwind.config.js` ni `postcss.config.js`.
